# R-W Restaurant

A modern restaurant website built with React.

## Project Structure

```
walid-orabi/
├── src/             # React frontend
│   ├── Pages/
│   ├── Components/
│   └── Assets/
├── public/
└── package.json
```

## Setup Instructions

### Prerequisites
- Node.js (v14 or higher)

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the React development server:
   ```bash
   npm start
   ```
   App runs on `http://localhost:3000`

## Features

- **Responsive Design**: Mobile-friendly restaurant website
- **Modern UI**: Clean, professional design with smooth animations
- **Menu System**: Interactive menu with cart functionality
- **Contact Integration**: Contact form and location information

## Technologies Used

### Frontend
- React 18
- React Router DOM
- CSS3 with modern layouts
- Font Awesome icons
- Material-UI
- Bootstrap

## Available Scripts

- `npm start` - Runs the React app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run deploy` - Deploys to GitHub Pages

## Deployment

The app is configured for deployment to GitHub Pages.

1. Build the React app: `npm run build`
2. Deploy: `npm run deploy`

The app will be available at `https://walid-orabi.github.io/walid-orabi`
